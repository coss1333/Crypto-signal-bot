# Crypto Signal Bot (RSI/Volume/Breakout) → Telegram

**Что делает:** регулярно сканирует рынок криптовалют (через публичные API бирж ccxt), 
вычисляет набор индикаторов (RSI, полосы Боллинджера, всплеск объёмов, ATR%, пробой Дончиана, "сквиз") 
и отправляет в Telegram сигнал по монетам, которые с высокой вероятностью готовы к **резкому движению**.

## Быстрый старт (Windows/Mac/Linux)
1) Установите Python 3.10+
2) Распакуйте архив `crypto_signal_bot.zip`
3) В терминале: 
   ```bash
   cd crypto_signal_bot
   python -m venv .venv
   .venv/Scripts/activate   # Windows
   # source .venv/bin/activate  # macOS/Linux
   pip install -r requirements.txt
   ```
4) Скопируйте `config.example.env` → `.env` и заполните:
   - `TELEGRAM_BOT_TOKEN` — токен вашего бота
   - `TELEGRAM_CHAT_ID` — id чата/канала/группы
   - при необходимости поменяйте `TIMEFRAME`, `SCAN_INTERVAL_SEC`, `TOPN`, `ALERT_SCORE_THRESHOLD`
5) Запуск сканера:
   ```bash
   python main.py
   ```

## Логика сигналов (оценка 0..10)
- **RSI экстремы**: RSI(14) ≤ 25 (перепроданность) или ≥ 75 (перекупленность) — +1
- **Всплеск объёма**: z-score объёма ≥ 2.0 — +1
- **Пробой Дончиана(20)**: текущая цена > верхнего канала или < нижнего — +1
- **Сквиз/расширение**: узкие полосы Боллинджера → резкое расширение + объём — +1
- **Высокая волатильность**: ATR% (ATR/close) в верхнем квантиле — +1

Монеты сортируются по **Score**, в сигнал попадают те, у кого Score ≥ `ALERT_SCORE_THRESHOLD` 
(по умолчанию 3) — это гибрид **momentum + mean-reversion** эвристик.

## Настройка универса
- Если `.env` → `WATCHLIST` пуст, бот выберет **топ N по 24h объёму** USDT-пары на выбранной бирже.
- Можно задать вручную: `WATCHLIST=BTC/USDT,ETH/USDT,SOL/USDT`.

## Подсказки
- **Биржа**: по умолчанию `binance`. Также подходят `bybit`, `okx` (идентификаторы ccxt).
- **Таймфрейм**: начните с `1h`, затем попробуйте `15m` или `4h`.
- **Интервал сканирования**: `300` секунд (5 минут) — компромисс между актуальностью и лимитами API.

## Безопасность
- Бот использует только **публичные** эндпоинты биржи, без ключей. Ключи не нужны.
- Сообщения в Telegram отправляются через токен вашей учётки бота.

## Остановка
- Нажмите `Ctrl+C` в терминале, где запущен скрипт.
